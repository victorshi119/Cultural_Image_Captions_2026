# Guarani Caption Reference

## 1. Cross-reference prefixes (THE critical paradigm)

Active set (verb subject; intransitive activity / transitive agent):
| Person | Areal (V_, C_) | Aireal (i-stem) | Notes |
|---|---|---|---|
| 1SG | a- | ai- | a-guata "I walk" |
| 2SG | re- | rei- | re-mba'apo "you work" |
| 3 | o- | oi- | sg or pl; +hikuái for unambiguous PL |
| 1PL.INCL | ja- / ña- | jai- / ñai- | ña- before nasal root |
| 1PL.EXCL | ro- | roi- | (≠ portmanteau ro-) |
| 2PL | pe- | pei- | |

Inactive / "chendal" set (stative subject; transitive patient; possessor):
| Person | Form (oral / nasal) |
|---|---|
| 1SG | che- |
| 2SG | nde- / ne- |
| 3 | i- / ij- / iñ- / hi'- |
| 1PL.INCL | ñande- / ñane- |
| 1PL.EXCL | ore- |
| 2PL | pende- / pene- |

3rd-person inactive allomorphs: **i-** before C; **ij-** before unstressed V; **hi'-** before stressed V; **iñ-** before nasal base. ipo "his hand", ijao "his clothes", hi'áva "his hair", iñakã "his head".

Portmanteau (1→2): **ro-** = "I/we → you (sg)"; **po-** = "I/we → you (pl)". Rohayhu "I love you"; Pohendu "I listen to you (pl)".

3>1, 3>2: use inactive prefix on verb. chenupã "s/he hits me"; nderayhu "s/he loves you".

Imperative 2SG: **e-**. Egueru "Bring it!" Ehupi "Pick up!"

## 3. Negation circumfix

**nd- … -i** (oral) / **n- … -i** (nasal). After base ending in i/ĩ, suffix is **-ri**.

Prefix shapes by person:
- 1SG.ACT: **nda-** + a- → nd-a-…-i; ndakarúi "I don't eat", ndaikuaái "I don't know"
- 2SG.ACT: **nde-re-** (oral) / **ne-re-** (nasal); nderejoguái, neremañái
- 3.ACT: **ndo-** / **no-**; ndojoguái, nomañái
- 1PL.INCL: **ndaja-** / **naña-**
- 1PL.EXCL: **ndoro-** / **noro-**
- 2PL: **ndape-** / **nape-** (note: non-harmonic a)
- Inactive: **nda-/na-** + che-/nde-/i-/ñande-/ore-/pende-; ndachepochýi "I'm not angry"

NEG-future replaces -ta with **-mo'ã**: namba'apomo'ãi "I won't work".
NEG imperative: **ani** / **anína** + verb. Ani repoko cherehe "Don't touch me".
"yes" = heẽ ; "no" = nahániri ; "not yet" = ne'ĩra (with NON-negated verb).

## 4. Word order & syntax cheatsheet

- Pro-drop OK (subject especially). Ahecháma. = "I already saw it."
- Order flexible: SVO / VOS / VSO. Animate object usually post-V.
- **No copula.** Ndesy mbo'ehára. "Your mother (is a) teacher."
- Existential: oĩ "there is"; ime "be located"; iko "live/be in state".
- Yes/no Q: 2nd-position **=pa** or **=piko**. Ndépa Pablo? "Are you Pablo?"
- WH-Q: WH-word + =pa/=piko. Moõpa rehose? "Where do you want to go?"
- Direct/inverse on transitives: prefix tracks higher of 1>2>3 (active=agent, inactive=patient).
- Postpositions are enclitics: =pe (loc/dat) / =me (nasal); =gui (from); rehe (at); =gotyo (towards); =rupi (through); =ári (on); =guive (since).

## 5. Possession & relational t-/r-/h-

Many nouns have 3 forms: t- (no possessor, citation), r- (with possessor), h- (3rd-poss).
- t-óga "house (no poss)" → cheróga "my house", héra (=h-éra) "his name", nderóga "your house".
- t-esa/r-esa/h-esa "eye": nderesa "your eye", hesa "his eye".
- vaka ro'o "cow's meat (beef)"; t-o'o "flesh"; so'o "(eatable) meat".

## 8. Productive morphology

- **Causative**: mbo-/mby-/mbu- (oral) ~ mo-/my-/mu- (nasal). mongaru "feed" (mo-ngaru); mombe'u "tell".
- **Agent-demoting (passive-like)**: je- / ñe-. ojekuaa "it is known"; oñenupã "s/he is beaten".
- **Reciprocal**: jo- / ño-. ojohayhu "they love each other"; oñohenói "they call each other".
- **Plural (nouns)**: =kuéra / =nguéra (mostly animate). jaguakuéra "dogs"; mitãnguéra "children".
- **Diminutive**: -i/-mi. tape'i "small road/path".
- **Nominalizer**: -ha (place/agent); -kue/-ngue (post-stative "former"). akãngue "decapitated head".
- **Compound**: roots juxtapose; nasal tilde preserved on each nasal root. kuãirũ "ring" (kuã+irũ); avañe'ẽ "Guarani" (ava+ñe'ẽ).
- **Reduplication**: ~ marks plurality / iteration / intensity. overavera "lightning is flashing"; mokõimokõi "in pairs".

## 9. Critical reminders

Nasal harmony: a nasal vowel/consonant in the **root** triggers nasal allomorphs of prefixes/clitics LEFTWARD up to ~3 syllables (mb→m, nd→n, j→ñ, g→g̃, k→ng in =kuéra→=nguéra, =pe→=me). Negation circumfix: oral **nd-…-i** / nasal **n-…-i** (ending in i/ĩ → -ri).
