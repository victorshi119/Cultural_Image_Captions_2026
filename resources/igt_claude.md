# Guaraní (avañe'ẽ) — Caption pattern reference

## Legend
- 1SG/2SG/3/1PL.INCL/1PL.EXCL/2PL = persons; ACT = active series, INACT = inactive series
- POSSM = relational linker (`r-` after possessor, `h-` for 3); NPOSSM = unpossessed (`t-`)
- AGD `je-`/`ñe-` (agent-demoting); RECP `jo-`/`ño-`; MAKE1 causative `mbo-`/`mo-`
- FUT `-ta`; NEG circumfix `nd(a)- … -(r)i`; PROG `hína`; ADJZ `-va`; PL `=kuéra`/`=nguéra`
- =pe loc/dat/animate-DO; =gui from; =rehe/=re about/on; =ndive with; =gua of/from
- Nasal harmony at nasal stems: `mbo-→mo-`, `je-→ñe-`, `jo-→ño-`, `ja-→ña-`, `nde-→ne-`, `i-→iñ-`, `kuéra→nguéra`

---

## 1. Person prefixes

```
            ACTIVE          INACTIVE
1SG         a-              che-
2SG         re-             nde- / ne-
3           o- / oi-        i- / iñ- / h-
1PL.INCL    ja- / ña-       ñande- / ñane-
1PL.EXCL    ro-             ore-
2PL         pe-             pende- / pene-
1>2SG       ro-  (portmanteau)
1>2PL       po-  (portmanteau)
```

ACTIVE = subject of dynamic verbs and transitive subjects. INACTIVE = subject of stative/property predicates, object of transitives, possessor of nouns.

```
a-guata                      che-mandu'a                  i-porã
1SG.ACT-walk                 1SG.INACT-remembrance        3.INACT-beautiful
'I walk(ed)'                 'I remember(ed)'             's/he/it is beautiful'

re-ho                        nde-katupyry                 o-puka
2SG.ACT-go                   2SG.INACT-efficient          3.ACT-laugh
'you go/went'                'you are skilful'            's/he laughs'
```

Nasal harmony in prefix selection:

```
a-monda            n-a-monda-i              iñ-akã                ñane-r-asẽ
1SG.ACT-steal      NEG-1SG.ACT-steal-NEG    3.INACT-head          1PL.INCL.INACT-POSSM-cry
'I steal'          "I don't steal"          'his/her head'        'we cry'
```

Portmanteau (NEVER `a-…ndéve` for 'I…you'):

```
ro-h-ayhu                            po-h-ayhu
1>2SG-POSSM3-love                    1>2PL-POSSM3-love
'I/we love you (sg.)'                'I/we love you (pl.)'
```

## 2. Possession

### 2a. Inactive prefix on noun = possessor

```
che-r-óga                    nde-po                       i-juru
1SG.INACT-POSSM-house        2SG.INACT-hand               3.INACT-mouth
'my house'                   'your hand'                  'his/her mouth'

iñ-akã                       ñane-r-etã                   pene-memby
3.INACT-head                 1PL.INCL.INACT-POSSM-country 2PL.INACT-child.of.woman
'his/her head'               'our country'                'your (pl.) children'
```

Biform/triform roots take `r-` after a possessor, `h-` for 3, `t-` unpossessed. Common: óga/róga/hóga 'house'; túva/ru/itúva 'father'; tesa/resa/hesa 'eye'; tete/rete/hete 'body'; tova/rova/hova 'face'; tembireko/rembireko/hembireko 'wife'; ta'ýra/ra'y/ita'ýra (man's) 'son'; tajýra/rajy/itajýra (man's) 'daughter'.

### 2b. Possession by juxtaposition: POSSESSOR POSSESSED

```
María ajaka                  Lili r-ague                  vaka r-o'o
María basket                 Lili POSSM-hair              cow POSSM-flesh
'María's basket'             'Lili's hair'                'cow's meat / beef'

ñandu r-enimbo               jasy r-enyhẽ                 yvága r-ovy
spider POSSM-thread          moon POSSM-full              sky POSSM-blue
'spider's web'               'full moon'                  'blue sky'
```

## 3. TAM and negation

```
o-sẽ-ta                              re-h-ai hína
3.ACT-go.out-FUT                     2SG.ACT-POSSM3-write PROG
's/he will go out'                   'you are writing'

a-ha-ma                              o-mano-mba
1SG.ACT-go-already                   3.ACT-die-all
'I already went'                     'they all died'

a-ha-va'ekue                         nd-a-japo-i
1SG.ACT-go-PAST                      NEG-1SG.ACT-make-NEG
'I went (long ago)'                  "I didn't do it"

nda-che-pochy-i                      n-a-mba'apo-mo'ã-i
NEG-1SG.INACT-anger-NEG              NEG-1SG.ACT-work-NEG.FUT-NEG
'I am not angry'                     'I will not work'

ani re-ho
NEG.IMP 2SG.ACT-go
'do not go'
```

NEG allomorphs: `nd-` before V, `nda-` before C, `n-` before nasal. `-i` → `-ri` after i/glottal-final stems.

## 4. Postpositions / case enclitics

```
=pe       at, in, to; animate direct object & dative
          a-ha-ma kuehe pe mbo'ehao=pe
          1SG.ACT-go-already yesterday MED.SG school=in
          'I already went to school yesterday'

=gui      from (ablative)
          a-ha Luque=gui Paraguay=pe
          1SG.ACT-go Luque=from Paraguay=in
          'I go from Luque to Asunción'

=rehe / =re   about, at, on (object of verbs of looking, remembering, caring)
          o-ma'ẽ pe t-ape=re
          3.ACT-look.at MED.SG NPOSSM-road=at
          's/he looks at that road'

=ndive    with (comitative)
          e-ho jarýi=ndive
          IMP-go grandmother=with
          'go with grandmother'

=gua      of, from (origin / belonging)
          jagua óga=gua                'dog of the house'

=icha     like, as
          nde-sy i-porã yvoty-icha     'your mother is beautiful like a flower'
          2SG.INACT-mother 3.INACT-beautiful flower-as

=ári      upon, on top of
          kavaju='ári                  'on horseback'
          horse=upon

=guarã    for (benefactive)
          a-japo che-sy=pe=guarã       'I made (it) for my mother'
          1SG.ACT-make 1SG.INACT-mother=in=for

=kuéra / =nguéra   plural clitic
          mitã=nguéra 'children';  arai=kuéra 'clouds'
```

## 5. Compounding & modification

Order: HEAD MODIFIER. Adjectives, colors, and nominal modifiers follow the head. The `r-` linker shows up on biform/triform roots after a modifier-host.

```
mitã+porã                    auto tuja                    kuña mbarete
child+beautiful              car old                      woman strong
'pretty child'               'old car'                    '(a) strong woman'

yvága r-ovy                  panambi h-ovy                t-esa r-ovy
sky POSSM-blue               butterfly POSSM3-blue        NPOSSM-eye POSSM-blue
'(a) blue sky'               '(a) blue butterfly'         'blue eye(s)'

py+ao                        kuã+irũ                      tape puku
foot+clothes                 finger+friend                road long
'sock(s)'                    'ring'                       '(a) long road'
```

Predicative ATTRIBUTE-on-N uses inactive prefix:

```
kuña i-mbarete               panambi i-pepo
woman 3.INACT-strong         butterfly 3.INACT-wing
'the woman is strong'        'butterflies have wings'
```

Demonstratives precede the noun: `ko` (this), `pe` (that), `amo` (that yonder), `ko'ã` (these), `umi` (those).

```
ko jagua          pe tape puku           amo táva           umi guyra
this dog          that long road         that town yonder   those birds
```

## 6. Caption-shaped templates

```
T1.  SUBJ V-s (intransitive)
     kuarahy o-vera                          'the sun is shining'
     sun     3.ACT-shine

T2.  SUBJ V-s OBJ (3>3 transitive)
     Kola oi-nupã i-mbarakaja                'Kola beats his cat'
     Kola 3.ACT-beat 3.INACT-cat

T3.  SUBJ V-s OBJ to RECIPIENT=pe
     pe karai o-me'ẽ so'o i-tajýra=pe        'that man gives meat to his daughter'
     MED gentleman 3.ACT-give meat 3.INACT-daughter=in

T4.  SUBJ has OBJ
     i-túva o-guereko peteĩ jagua'i           'his father has a little dog'
     3.INACT-father 3.ACT-have one dog-DIM

T5.  SUBJ is at LOC=pe
     pe-iko kokue=pe                          'you live in the countryside'
     2PL.ACT-be countryside=in

T6.  SUBJ is ATTRIBUTE  (stative → INACTIVE prefix)
     kuña i-katupyry                          'the woman is skilful'
     woman 3.INACT-skilful

T7.  SUBJ has BODY-PART (inalienable)
     panambi i-pepo                           'butterflies have wings'
     butterfly 3.INACT-wing

T8.  SUBJ V-s about/at TARGET=re/rehe
     ha'e o-ma'ẽ pe t-ape=re                  's/he looks at that road'
     s/he 3.ACT-look.at MED NPOSSM-road=at

T9.  SUBJ V-s with INSTR=pe
     typycha=pe o-ñe-mo-sẽ                    'he was kicked out with a broom'
     broom=in 3.ACT-AGD-MAKE1-go.out

T10. SUBJ V-s from SOURCE=gui to GOAL=pe
     a-ha Luque=gui Paraguay=pe               'I go from Luque to Asunción'
     1SG.ACT-go Luque=from Paraguay=in

T11. SUBJ V-s with COMPANION=ndive
     e-ho Peru jarýi=ndive                    'go with Peter's grandmother'
     IMP-go Peru grandmother=with

T12. There is/are X (existence)
     h-eta mymba                              'there are many animals'
     POSSM3-numerous animal

T13. Equative SUBJ PRED-NOUN (no copula)
     nde-sy mbo'e-hára                        'your mother is a teacher'
     2SG.INACT-mother teach-NMLZ.AG

T14. N i-ADJ-va as relative clause
     peteĩ porandu i-porã-va                  '(a) beautiful question'
     one question 3.INACT-beautiful-ADJZ
```

## 7. Pitfalls

**P1. Active vs. inactive on stative predicates.** Properties, sensations, and inalienable possessions take INACTIVE.
```
✓ che-vare'a         ✗ *a-vare'a            'I am hungry'
✓ i-porã             ✗ *o-porã              'it is beautiful'
```

**P2. Nasal harmony.** Oral allomorph at a nasal stem is ungrammatical.
```
✓ iñ-akã             ✗ *i-akã
✓ ñane-r-asẽ         ✗ *ñande-r-asẽ
✓ pene-memby         ✗ *pende-memby
```

**P3. 1→2 portmanteau.** Use `ro-`/`po-`, not `a- … ndéve`.
```
✓ ro-h-ayhu          ✗ *a-h-ayhu ndéve      'I love you'
```

**P4. Missing `r-`/`h-` linker on biform/triform nouns.** Required after any possessor.
```
✓ che-r-óga          ✗ *che-óga
✓ vaka r-o'o         ✗ *vaka o'o
✓ jasy r-enyhẽ       ✗ *jasy enyhẽ
```

**P5. Spanish-style adjective-before-noun.** Modifiers FOLLOW the head.
```
✓ mitã porã          ✗ *porã mitã           'pretty child'
✓ tape puku          ✗ *puku tape           'long road'
```

**P6. Dropping `=pe` on animate direct objects.**
```
✓ ai-kuaa amo kuña=me        ✗ *ai-kuaa amo kuña     'I know that woman'
✓ a-h-ayhu jagua=kuéra=pe    ✗ *a-h-ayhu jagua=kuéra 'I love the dogs'
```

**P7. Half-circumfix negation.** Need BOTH `nd(a)-` AND `-i`. Future negation needs `n(d)- … -mo'ã-i`.
```
✓ nd-a-japo-i        ✗ *nd-a-japo / *a-japo-i
✓ n-a-mba'apo-mo'ã-i ✗ *nd-a-mba'apo-i      (for future 'I will not work')
```

**P8. `=pe` (locative) vs. `=re/rehe` (oblique).** Verbs of looking, thinking, remembering, caring take `=re/=rehe`, not `=pe`.
```
✓ o-ma'ẽ tape=re             ✗ *o-ma'ẽ tape=pe         's/he looks at the road'
✓ che-mandu'a nde=rehe       ✗ *che-mandu'a ndéve      'I remember you'
```
